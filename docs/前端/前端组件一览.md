# 前端组件一览

## 1. weiyang-background

未央背景图标。在页面右下侧生成一个未央图标，fixed定位，z-index=-15。

### 可用属性

无

### 使用示例

```html
<template>
	<view>
		<weiyang-background></weiyang-background>
	</view>
</template>
```
